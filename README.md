# Tiny Test Project

**Key features:**
- Architecture: MVI
- DI: Hilt
- UI: Jetpack Compose
- Concurrency: Coroutines
- Unit test: Mockk
- UI test: Jetpack Compose
- Local Db: Room
- Remote client: Ktor

Aside from creating the project and coding it to be functional, I've implemented some Unit and UI tests as examples.
